      AWD攻防赛-攻略

      somenzz 2020-10-21 AWD

      今年第一次参加网络安全攻防赛（AWD）。真的刺激，比刺激战场还刺激。

      最开始是预赛，通过预赛才能进入个人赛和团队赛，这里的团队赛就是AWD。

      预赛那会，很快完成理论赛和两道 ctf 的题目，感觉非常简单，自然进入复赛不在话下。

      一个月后的个人赛开始。也就是 10 月 20 日，个人赛的理论部分只有 30 分钟，有 80 个选择题，包括单选、多选和判断，基本是需要秒答的。后面是 3 个小时的 CTF 题目，一个小时过去了，一题也没有解出来，信息备受打击，比赛结束前，一道题目也没作出来，心态一度崩溃，开始怀疑人生。

      比赛结束后发现普遍反馈题目较难，80 多个人，有 50 多个都是一题没做出来，从结果来看，只要做出一题，就能获奖，于是也就不再纠结自己是否太笨。

      于是立即着手准备第二天的团队赛 AWD（攻防赛）。 我们三个分讨论策略，编写脚步，一直搞到晚上 12 点，从任务分工，网站及数据库备份恢复、目录监控、访问日志记录、安全加固、端口扫描、批量攻击等方面准备的非常充分，信心满满的参加第二天的团队赛。

      团队赛一开始进行的还算顺利，备份工作已经完成，第一轮的服务检测有两支队伍因服务宕机而被扣 100 分，我们暗自庆幸，结果在安全加固时因文件目录问题导致服务无法，第二轮检测是被系统检测到服务宕机被扣 100 分，排名一度成为倒数第一，心态再次失衡，好在团队相互鼓励，不气馁，做好检测和备份恢复工作，准备防守：发现入侵立刻将其上传文件删除，发现服务宕机立即恢复，发现漏洞立即修补，靠着这种方式我们躲过了来自其他队伍的攻击，而不少队伍因为防守不够导致服务宕机被多次扣 100 分，渐渐的我们排名开始拉升。最后发现 Think PHP 5.0.23 版本的远程代码执行漏洞开始攻击其他队伍，拿到 16 次 flag，最终以第九名的成绩荣获团队三等奖。

      这里参照前人经验对 AWD 比赛准备总结如下。

      # 一、AWD介绍

      AWD：Attack With Defence，即攻防对抗，比赛中每个队伍维护多台服务器（一般两三台，视小组参赛人数而定），服务器中存在多个漏洞（web层、系统层、中间件层等），利用漏洞攻击其他队伍可以进行得分，加固时间段可自行发现漏洞对服务器进行加固，避免被其他队伍攻击失分。

      攻击有两种方式：一是拿到对方的 flag，自己得分，对方失分；二是修改对方的项目文件，导致其服务宕机，对方严重丢分。显然方式二略有难度。攻击成功之后就可以编写脚本批量攻击，更快的获取分数，或让对方丢分。

      防也有两种方式：一是修补漏洞，防止对方拿到 flag，二是服务宕机后立马恢复，显然方式二要简单的多。

      1. 各队一般都有一个初始分数，flag 一旦被其他队伍拿走，该队扣除一定积分，得到 flag 的队伍加分（本次为 10 分）；
      2. 一般分配 Web 服务器（多数为 PHP 环境），服务器（多数为Linux）某处存在 flag（一般在根目录下）；
      3. 可能会提供一台流量分析虚拟机，可以下载流量文件进行数据分析（较少提供）；
      4. flag 在主办方的设定下每隔一定时间刷新一轮；
      5. 一般每个队伍会给一个低权限用户，非 root 权限；
      6. 主办方会定时（15分钟）对每个队伍的服务进行检查，服务器宕机扣除本轮 flag 分数（本次为 100 分）

      # 二、比赛分工

      1、线下赛一般 3 人，2 人攻 1 人守，在此情况下攻击比防守得分来的快； 也可一人维护一台靶机，同时进行攻防。 2、比赛靶机一般都在同一个 B 段，可提前扫描目标网段中存活靶机，方便实施攻击。 如使用御剑高速 TCP 全端口扫描工具、httpscan，只需要扫描 80 端口就行（一般 web 为 80 端口，特殊情况自行设置）。

      # 三、套路

      # 第一时间备份

      为什么备份？ 因为一旦别人攻进来，删除了你的项目关键文件导致服务崩溃，再也无法恢复，就会一直被扣分，这样基本就出局了。如果做了备份，就可以立即恢复至原状态。

      比赛开始第一时间备份，备份网站目录及数据库，一般在 /var/www/html 目录，网站复制一份或直接 scp 复制到本地；

      mysql 数据库备份则执行以下命令：

      mysqldump -h主机名  -P端口 -u用户名 -p密码 数据库名 > 文件名.sql
      1

      恢复的话，直接使用mysql 命令执行脚本即可：

      mysql -h主机名  -P端口 -u用户名 -p密码 数据库名 < 文件名.sql
      1

      1、网站项目下载到自己电脑之后，再复制一份作为初始备份，万一修改错了导致服务宕机可以恢复初始状态， 2、用 D 盾查杀存在的后门，要审计下有没有高危命令执行函数，进行后期加固，以及有空余时间或专门安排一个人审计下基础漏洞用来做攻击。命令执行函数 exec()、passthru()、system()、 shell_exec()、popen() 等; 代码执行函数：eval()、assert()、preg_repace()、uasort() 等；文件包含函数 include() 、require() 等，用正则匹配 grep -r "@eval" /www/，找到后注释掉。 3、加固后的项目再传回靶机，这样别人就无法攻进来了。

      常见的 web 查杀工具：https://bypass007.github.io/Emergency-Response-Notes/Summary/第3篇：常见的Webshell查杀工具.html

      # 改掉弱口令

      大多数情况下，所有队伍的Web后台、phpmyadmin等服务的管理密码都一样，立马快速检查修改自己密码，并利用此进行攻击。一般默认密码为 admin/admin, admin/123456, test/test，如果被其他队伍改了那就gg了。

      # 加固（谨慎操作）

      加固就是在 php 文件插入一段代码，对请求参数做检测，屏蔽一些不安全的请求，如 sql注入中的 select 检测，检测不可见字符造成的截断和绕过效果，攻击关键字检测等功能，具体可以看下面的 php 代码：add_security.php

      
<?php
// highlight_file(__FILE__);LS /Applications/MAMP/htdocs/awd/awd_sec.php

// error_reporting(E_STRICT);
// ini_set('display_errors', "On");

/*
获取http请求头并写入数组
*/
function get_all_headers() {
    $headers = array();
    foreach ($_SERVER as $key => $value) {
        if (substr($key, 0, 5) === 'HTTP_') {
            $headers[$key] = $value;
        }
    }
    return $headers;
}
/*
检测不可见字符造成的截断和绕过效果，注意网站请求带中文需要简单修改
*/
function filter_invisible($str) {
    for ($i = 0; $i < strlen($str); $i++) {
        $ascii = ord($str[$i]);
        if ($ascii > 126 || $ascii < 32) { //有中文这里要修改
            if (!in_array($ascii, array(
                9,
                10,
                13
            ))) {
                write_attack_log("interrupt");
            } else {
                $str = str_replace($ascii, " ", $str);
            }
        }
    }
    $str = str_replace(array(
        "`",
        "|",
        ";",
        ","
    ) , " ", $str);
    return $str;
}
/*
检测网站程序存在二次编码绕过漏洞造成的%25绕过，此处是循环将%25替换成%，直至不存在%25
*/
function filter_0x25($str) {
    if (strpos($str, "%25") !== false) {
        $str = str_replace("%25", "%", $str);
        return filter_0x25($str);
    } else {
        return $str;
    }
}
/*
攻击关键字检测，此处由于之前将特殊字符替换成空格，即使存在绕过特性也绕不过正则的\b
*/
function filter_attack_keyword($str) {
    if (preg_match("/select\b|insert\b|update\b|drop\b|delete\b|dumpfile\b
|outfile\b|load_file|rename\b|floor\(|extractvalue|updatexml|name_const|m
ultipoint\(/i", $str)) {
        write_attack_log("sqli");
    }
    if (substr_count($str, $_SERVER['PHP_SELF']) < 2) {
        $tmp = str_replace($_SERVER['PHP_SELF'], "", $str);
        if (preg_match("/\.\.|.*\.php[35]{0,1}/i", $tmp)) {
            write_attack_log("LFI/LFR");;
        }
    } else {
        write_attack_log("LFI/LFR");
    }
    if (preg_match("/base64_decode|eval\(|assert\(/i", $str)) {
        write_attack_log("EXEC");
    }
    if (preg_match("/flag/i", $str)) {
        write_attack_log("GETFLAG");
    }
}
/*
简单将易出现问题的字符替换成中文
*/
function filter_dangerous_words($str) {
    $str = str_replace("'", "‘", $str);
    $str = str_replace("\"", "“", $str);
    $str = str_replace("<", "《", $str);
    $str = str_replace(">", "》", $str);
    return $str;
}
/*
获取http的请求包，意义在于获取别人的攻击payload
*/
function get_http_raw() {
    $raw = '';
    $raw.= $_SERVER['REQUEST_METHOD'] . ' ' . $_SERVER['REQUEST_URI'] . ' ' . $_SERVER['SERVER_PROTOCOL'] . "\r\n";
    foreach ($_SERVER as $key => $value) {
        if (substr($key, 0, 5) === 'HTTP_') {
            $key = substr($key, 5);
            $key = str_replace('_', '-', $key);
            $raw.= $key . ': ' . $value . "\r\n";
        }
    }
    $raw.= "\r\n";
    $raw.= file_get_contents('php://input');
    return $raw;
}
/*
这里拦截并记录攻击payload
*/
function write_attack_log($alert) {
    // echo "write_attack_log";
    date_default_timezone_set("Asia/Shanghai");
    $rip = $_SERVER["REMOTE_ADDR"];
    $data = date("Y/m/d H:i:s"). " REMOTE_ADDR:" . $rip . " -- " . $alert . "]" . "\r\n" . get_http_raw() . "\r\n\r\n";
    $ffff = fopen('/Applications/MAMP/htdocs/awd/log.txt', 'a') or die("Unable to open file!"); //日志路径
    fwrite($ffff, $data);
    fclose($ffff);
    if ($alert == 'GETFLAG') {
        header("location:http://172.16.9.2/");
    } else {
        sleep(15); //拦截前延时15秒
    }
    exit(0);
}

for root,dirs,files in os.walk(project_dir)
    for file in files:
        abs_path = os.join(root,file)
        if os.path.isfile(abs_path) and re.match(regex,os.path.splitext(ppp)[1]):
            filepaths_will_be_modified.append(abs_path)
     
for file_will_modified in filepaths_will_be_modified:
    if file_will_modified!= security_php_path:
        insert_line = '@include_once("'+security_php_path+'");\n';
        new_content = ''
        old_content = ''
        php_flag = False    #avoid the html page content
        is_insert = False

        with open(file_will_modified,"r") as reader:
            for line in reader:
                old_content += line
                if line.startswith('<?php'):
                    php_flag = True
                    new_content += line 
                    continue
                if line.replace("\n","").replace(" ","").replace("\r","")=="":
                    continue
                if not line.startswith('use') and not line.startswith('namespace') and not line.startswith('/') and not line.startswith(' *') and php_flag and not is_insert:
                    new_content += insert_line
                    is_insert = True
                new_content += line
 
        if is_insert:
            #备份
            with open(file_will_modified + '.bak',"w") as writer:
                print('backup file:',file_will_modified+'.bak')
                writer.write(old_content)
            #写入新内容
            with open(file_will_modified,"w") as writer:
                print('modified',file_will_modified)
                writer.write(new_content)

      # 监控文件系统

      如果有人上传木马等需要立即删除，否则很容易被人拿到 flag，这里提供一段文件系统监控程序可以立即删除新增的文件，防止上传文件。

      import time
from log import logger
from watchdog.observers import Observer
from watchdog.events import FileSystemEventHandler

do_not_delete = 'wjrcb'
project_path = './'

def cat_file(path):
    if os.path.isfile(path):
        with open(path) as r:
            logger.info('文件内容如下：\n'+ r.read())

def delete_file(path):
    if os.path.isfile(path):
        logger.info(f"deleting {path} ...")
        os.remove(path)
        logger.info("done")


class MyHandler(FileSystemEventHandler):

    def on_created(self,event):
        logger.info(f"find new {event.src_path}:")
        cat_file(event.src_path)
        if not (do_not_delete in event.src_path):
            delete_file(event.src_path)

    def on_modified (self,event):
        if os.path.isfile(event.src_path):
            logger.info(f"find modified {event.src_path}:")
            cat_file(event.src_path)


if __name__ == "__main__":
    path = sys.argv[1] if len(sys.argv) > 1 else project_path
    event_handler = MyHandler()
    observer = Observer()
    observer.schedule(event_handler, path, recursive=True)
    observer.start()
    try:
        while True:
            time.sleep(1)
    except KeyboardInterrupt:
        observer.stop()
    observer.join()

    return isinstance(value, (list, tuple, set))

# 获取Unicode编码
def getUnicode(value, encoding=None, noneToNull=False):
    if noneToNull and value is None:
        return NULL
    if isListLike(value):
        value = list(getUnicode(_, encoding, noneToNull) for _ in value)
        return value
    if isinstance(value, unicode):
        return value
    elif isinstance(value, basestring):
        while True:
            try:
                return unicode(value, encoding or UNICODE_ENCODING)
            except UnicodeDecodeError, ex:
                try:
                    return unicode(value, UNICODE_ENCODING)
                except:
                    value = value[:ex.start] + "".join(INVALID_UNICODE_CHAR_FORMAT % ord(_) for _ in value[ex.start:ex.end]) + value[ex.end:]
    else:
        try:
            return unicode(value)
        except UnicodeDecodeError:
            return unicode(str(value), errors="ignore")

# 计算机文件MD5值
def calcMD5(filepath):
    try:
        with open(filepath,'rb') as f:
            md5obj = hashlib.md5()
            md5obj.update(f.read())
            hash = md5obj.hexdigest()
            return hash
    except Exception, e:
        print u'[!] getmd5_error : ' + getUnicode(filepath)
        print getUnicode(e)
        try:
            ORIGIN_FILE_LIST.remove(filepath)
            FILE_MD5_DICT.pop(filepath, None)
        except KeyError, e:

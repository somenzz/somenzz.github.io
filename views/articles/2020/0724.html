理解Python中的 args 和 *kwargs

somenzz 2020-07-09 Python

我发现大多数新的 Python 程序员都很少使用 *args 和 **kwargs 参数。那他们是什么？

首先，它们是个函数参数，只不过前面多了个一个或两个星号，表达的不限参数个数的意思，一个星号表示列表（list），两个星号表示字典（dict）

举个例子就明白了：

>>> def test_var_args(f_arg, *argv):
...     print("first normal arg:", f_arg)
...     for arg in argv:
...         print("another arg through *argv:", arg)
...
>>> test_var_args('first', 'python', 'eggs', 'test')
first normal arg: first
another arg through *argv: python
another arg through *argv: eggs
another arg through *argv: test
>>> v_list=[1,2,3,4,5,6]
>>> test_var_args('first', *v_list)
first normal arg: first
another arg through *argv: 1
another arg through *argv: 2
another arg through *argv: 3
another arg through *argv: 4
another arg through *argv: 5
another arg through *argv: 6

在传入参数时，一个星号可以将一个列表转化为顺序传入的参数，也叫做解包，如：test_var_args('first', *v_list), 同样的，两个星号可以将一个字典转化为关键字参数。如下所示：

>>> def greet_me(**kwargs):
...     for key, value in kwargs.items():
...         print("{0} = {1}".format(key, value))
...
>>> greet_me(name="first")
name = first
>>>
>>> kwargs = {"arg3": 3, "arg2": "two", "arg1": 5}
>>> greet_me(**kwargs)
arg3 = 3
arg2 = two
arg1 = 5

同时使用 *args **kwargs 普通参数时是有顺序的， 则顺序为

some_func(fargs, *args, **kwargs)
1

此外，还有一个问题，如果直接传入一个字典会，这属于 Python 的引用传递，函数可以改变传入的字典的内容，如果采用 ** 这种方法传入一个字典，就不会改变原有字典的内容，示例代码如下：

def fun1(v_dict):
    v_dict.update({'status':"changed"})
    return v_dict

def fun2(**v_dict):
    v_dict.update({'status':"changed"})
    return v_dict


v_dict = {'status':'origin'}
print("v_dict",v_dict)
new_dict = fun1(v_dict)
print("after execute fun1:")
print("v_dict",v_dict)
print("new_dict",new_dict)

v_dict = {'status':'origin'}
print("v_dict",v_dict)
new_dict = fun2(**v_dict)
print("after execute fun2:")
print("v_dict",v_dict)
print("new_dict",new_dict)

                          别问，问就是精品

                          somenzz 2020-09-30 Mac App

                          在某次系统升级之后，我的 MacBook Pro 每隔三五天会随机出现五国问题（也就相当于 Windows 的蓝屏），然后自动重启，虽然一分钟内能恢复现场，但内心仍有些不爽，如果正在调试程序，突然重启很影响效率，毕竟周围同事的 Mac 从来不需要重启。

                          作为一个技术控，怎么能容忍这个，我决定搞清楚原因。

                          关闭了所有自动启动项，确保没有任何软件自动启动，然后记录了每一次重启时自己正在使用的 app，发现每次都不一样，可以排除是某个具体的 app 导致的，大概率是系统问题，然后看了崩溃报告，有个 watchdog 模块出了问题，网上搜索了一番，也有人遇到此类情况，我预感是要重装系统了。于是先打了 apple 的客服电话咨询（之前从未打过 apple 客服），客服小姐姐不但声音好听，还懂技术，很快就 get 到问题重点，是我遇到过最完美的客服，不得不说 苹果连客服都做的无可挑剔，她建议我备份数据、抹掉磁盘、重新安装 MacOS，如果问题仍然出现，那就很可能是硬件问题，需要维修。我听从其建议重新安装了 macOS Catalina 10.15.6，目前已经一个多月没有出问题了，五国问题大概率是解决掉了。

                          抹掉重来，虽说折腾，但也不失为一种断舍离的生活方式，和过去的杂乱无章说再见，那些可有可无的应用程序和文件自然会消失，高度依赖的 app 自然会再次安装，这些重新回来的 10 个 App，别问，问就是精品，可以让你的 Mac 释放巨大的生产力，好用到根本停不下来。

                          # 1、Alfred：一切为了让效率更进一步

                          Alfred 是 macOS 上必备的应用程序，几乎每一个实用 mac 的用户都会安装它。可通过热键，关键字，文本扩展等功能提高效率。搜索 Mac 和网络，还可以通过自定义操作来控制 Mac 来提高生产力。网上的教程非常之多，这里抛砖引玉，如果还没安装的话，先看看网上的教程，如果你也注重效率的话， 相信你最终必然会安装。

                          我最常用的情形：虚拟机中的程序报错了，但报错信息无法复制到 mac，但又不想切出去打开浏览器搜索，因为自己记不住某些单词，怎么快速搜索呢，直接 option + 空格，会再报错界面会弹出一个输入框，可以一遍看报错信息，一边输入，完成后回车自动打开指定的搜索引擎搜索结果页面，非常高效。

                          还有很多非常高级的功能，详细功能见官方网站。

                          官方网站：https://www.alfredapp.com/

                          注意，免费用户只能使用其 Features 中的功能，即基本搜索和快速启动应用等功能，其实这已满足非重度使用者日常需求，若要使用 Workflows (即自定义插件的工作流)，则需要购买 Powerpack 才能激活此功能。

                          本人强烈建议读者使用正版。若要试用，请自行搜索破解版，或访问文末的资源分享网站。

                          # 2、iShot：截图，录屏信手拈来，所见即所得。（免费，app store 可直接安装）

                          Mac 自带的截图工具快捷键是 control+command+shift+4，虽然也好用，但是无法在截图中添加文字、画圈等操作，Snipaste 虽然有截图中添加文字、画圈等操作，但是无法截长图，最完美的就是 iShot，还可以聚焦截图中的某处，加强效果，截长图也是轻而易举。better365 出品的软件都不错，还免费，可谓良心企业，用爱发电。

                          安装方式：可在 Mac App Store 中搜索下载，也可官网 https://www.better365.cn/apps.html 下载安装，这里的 app 都很实用，小而美的精品软件，最重要的是全部免费！

                          # 3、iPic：Markdown 图床、文件上传工具。（免费，app store 可直接安装）

                          有了图床神器 iPic，不论屏幕截图、还是复制图片，都可以自动上传、保存 Markdown 格式的链接，直接粘贴插入，够懒人吧？

                          使用 Hexo 或 WordPress、写博客、在公众号发文章、在知乎讨论、在豆瓣灌水、在论坛发帖、跨境做外贸电商，iPic 带给你从未有过的插图体验。

                          当然，除了图片，你可以上传普通文件，上传方式与图片完全相同。

                          上传方式：

                          1、拖拽图片上传。拖动是比较好玩的一种上传方式。只要将图片拖到菜单栏的 iPic 图标上，松手后就可以自动上传。

                          2、使用服务上传图片。在 Finder 中使用服务 上传也是很高效的方式。只要在图片上右击、然后选择服务中的 使用 iPic 上传即可。

                          3、复制图片后上传。iPic 会自动监测剪切板的变化，当复制图片后，该图片会出现在 iPic 菜单中待上传区域。如果需要上传，点击菜单中该图片即可。手动上传比较适合临时上传少量图片。

                          除了手动点击菜单，还可以使用快捷键 Command + Shift + U 上传。当然，可以在偏好设置中修改此快捷键。

                          # 4、Paste 复制和粘贴更智能

                          Paste是 Mac，iPhone 和 iPad 上全球最受欢迎的剪贴板管理器。它存储您复制的所有内容，并使剪贴板在所有设备上井井有条。

                          比如说你复制了一些文本1，然后又复制了文本2，这个时候你又想粘贴文本1，通常情况下你需要找找文本1所在的位置再去复制一遍，如果用了 Paste，直接按 shift+command+v 会在底部弹出面板，选择那个文本1即可粘贴至需要的地方，极大的提高了效率。复制和粘贴是使用最频繁的功能，每个环节哪怕节省 2 秒，累积下来也是非常可观的。

                          app store 上可以直接下载，要付费 70 块左右，网上也可以找到破解资源，如果找不到的话，可访问文末的资源分享网站。

                          # 5、Iterm2：终端利器。（免费，官网下载直接安装）

                          iTerm2 是 MAC 下最好的终端工具，再配合 Oh My Zsh，你的终端从未如此完美。

                          我主要使用 iTerm2 的分屏功能（比 tmux 更完美）和以下功能点：

                          1、智能选中：

                          在 iTerm2 中，双击选中，三击选中整行，四击智能选中（智能规则可配置），可以识别网址，引号引起的字符串，邮箱地址等。选中即复制。即任何选中状态的字符串都被放到了系统剪切板中。

                          2、按住 ⌘ 键:

                          • 可以拖拽选中的字符串；
                          • 点击 url：调用默认浏览器访问该网址；
                          • 点击文件：调用默认程序打开文件；
                          • 点击文件夹：在 finder 中打开该文件夹；
                          • 同时按住option键，可以以矩形选中，类似于 vim 中的 ctrl v 操作；

                          免费，官网下载直接安装即可：

                          官方网站：https://www.iterm2.com

                          # 6、Parallels Desktop：Mac 最好用的虚拟机。

                          虚拟机就不多说了，程序员必备，Mac 下想玩 Windows 游戏的，也需要它，建议官网下载试用一下，觉得 OK，再付费或找资源。

                          官方网站：https://www.parallels.com/products/desktop/

                          强烈建议读者使用正版。若要试用，请自行搜索破解版，或访问文末的资源分享网站。

                          # 7、Cinch： 拖拽窗口管理。

                          Windows 7 窗口管理新功能 Aero Snap 方便有用，Mac 下则可用 Cinch 实现相同功能。软件将屏幕左右和上方边缘变为热区，将窗口移动进去可相应让窗口填满左/右半屏，向上则为最大化。Cinch 每次启动都会提示购买，你可以每次忽略提示坚持全功能免费使用。

                          强烈建议读者使用正版。若要试用，请自行搜索破解版，或访问文末的资源分享网站。

                          # 8、1password：密码管理器。

                          很多网站都有复杂密码的要求，这样给记忆带来沉重的负担，结果就是经常忘记密码，1password 应运而生，是存储和使用强密码的最简单方法。一键即可安全登录网站、填写表单。

                          官方网站：https://1password.com/zh-cn/

                          密码类的软件强烈建议读者使用正版。若要试用，请自行搜索破解版，或访问文末的资源分享网站。

                          # 9、小番茄：专注，让事情变得简单。（免费，app store 可直接安装）

                          基于番茄工作法，给任务设定一个执行时间，专注任务，直到时间结束，这收获的不仅仅是效率，还会有意想不到的成就感。

                          收集和管理所有事情。你可以用它来制定工作学习计划，记录购物清单，设置生日提醒或者用来安排日程，生活工作中所有的大小事，小番茄都能帮你轻松搞定。

                          统计历史数据。帮您分析整体的工作时间和任务完成情况，每天有多少时间在工作以及工作的时间都花到哪些项目，让时间有迹可循。

                          随时随地访问。无论你是在手机，电脑还是平板上，都能随时随地的记录任务或者开启一个番茄。

                          官方网站：https://www.focustodo.cn/?lang=zh_CN

                          # 10、Xmind：思维导图，思如泉涌 ，成竹在图。

                          XMind 是一个全功能的思维导图和头脑风暴软件，为激发灵感和创意而生。作为一款有效提升工作和生活效率的生产力工具，受到全球百千万用户的青睐。

                          我觉得它最好用的思维导图软件，没有之一，画图便捷，可以非常方便的导出 pdf、png、markdown、word、excel等主流格式。

                          官方网站：https://www.xmind.cn

                          本人强烈建议读者使用正版。若要试用，请自行搜索破解版，或访问文末的资源分享网站。
